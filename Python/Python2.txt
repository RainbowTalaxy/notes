第四章 操作列表

for dog in dogs:
	print(dog)
其中，dog 是新声明的变量（for语句里声明），dogs是已知列表
该循环可以遍历并打印 dogs 里的所有元素
for循环的内部语句需要全体缩进，for语句后面不能忘记加冒号

range(起,止) 函数可以生成一系列数字
比如：
for value in range(1,5):
	print(value)
输出为：
1
2
3
4
（不会打印5）
可以用 list() 函数将这一系列数字转换为列表，比如：numbers = list(range(1,6))
（其实 range函数 的返回值就是 list类型）
使用 range() 函数可以加入第三个参数来引入步长
比如：list(range(2,11,2))
生成的列表为：[2, 4, 6, 8, 10]

min(数字列表) 可以找出列表中最小值
max(数字列表) 可以找出列表中最大值
sum(数字列表) 则可以算数列表数字的总和

可以用for语句直接构建一个数字列表，比如：
[value**2 for value in range(1,11)]
这条语句能生成一个从1到10的平方数的列表
这条语句也能称之为列表解析，列表解析是指用简单的代码来生成一个列表

切片示例：
alpha = ['A', 'B', 'C', 'D', 'E']
alpha[0:3]则是一个从 A 到 C 的列表
列表名[起:止] 为切片的一般使用，其中起止可以是负数
可以省略起止来表示 从头开始/一直到末尾
并且起止可以同时省略（ value[:] 等价于 value )

元组是不可修改的列表，用圆括号创建，比如：
alpha= ('A','B','C','D')
可以用 alpha[下标] 来访问元素
执行 print(alpha) 时，输出也为圆括号
如果要修改元组，则需要重新定义一个元素

在缩进的时候要避免制表符和空格混用
