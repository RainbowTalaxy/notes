Category注意事项
分类是用于给原有类添加方法的，不能添加属性
分类中的@property只会生成setter/getter方法，不会生成实现以及私有成员变量
可以在分类中原有类中.h中的属性
如果分类中有和原有类同名的方法，会调用分类中的方法
（最好在开发中不要这么写）
如果多个分类有同名的方法，调用该方法时执行谁的方法由编译器决定
（会执行最后一个编译的方法）

查找编译器编译循序的方法：
点击左栏中的项目 -> Build Phases -> Compile Sources

方法的调用顺序：分类 -> 本类 -> 父类

匿名分类（类扩展 class extension)：
Extension 是 Category 的一个特例，但可以扩充一些成员变量和方法

格式：
@interface 类名()
@end
