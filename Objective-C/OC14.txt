Block基本概念：
Block是iOS中一种比较特殊的数据类型
是苹果官方特别推荐使用的数据类型
应用场景 - 动画 、多线程、集合遍历 、网络请求回调

作用：
保存一段代码，可以在恰当的时间取出来调用
功能类似于函数和方法，但强于函数与方法

指向函数的指针：
返回值 ( * xxx ) ( 形参列表 ) ;
xxx = 函数名 ;

block和函数一样，可以没有返回值，没有形参

声明一个Block变量：
返回值 ( ^ xxx ) ( 形参列表 ) ;           //形参列表可以只写数据类型，不写名称
xxx = ^( 形参列表 ) {                        //如果没有形参，可以不写括号
	//代码 ...
} ;
( 也可以直接初始化Block变量 ）

调用Block变量：
xxx() ;

例如：
void ( ^printBlock ) ( int ) ;
roseBlock = ^( int num ) {
	for ( int i = 0 ; i < num ; ++i ){
		print("...") ;
	}
} ;
