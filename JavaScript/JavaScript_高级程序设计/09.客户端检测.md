# 9 客户端检测

## 能力检测
可以这样检测客户端（浏览器）有没有某种属性、函数或方法：
```js
if (query) {
    // 使用 query
}
```
或者更可靠的方式：
```js
function isSortable(obj) {
    return typeof obj.sort == "function"
}
```
不要根据能力检测来识别浏览器
